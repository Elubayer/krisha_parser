## Описание проекта

Парсер Krisha.kz на Python для сбора объявлений о продаже квартир.  
Собирает:

- `title` — название объявления  
- `price` — цена  
- `city` — город  
- `street` — улица  
- `rooms` — количество комнат  
- `square_m2` — площадь в м²  
- `floor` — этаж  
- `link` — ссылка на объявление  

Проект также содержит инструменты анализа и визуализации собранных данных.
